PluginInstance

PluginInstance

This is the model that enables the interaction with the plugin instances running on the network node pipelines. Please see the network documentation for a detailed description of the Plugin Instances.

A PluginInstance is defined by a config object, a set of tags and the instance working hours. Use the make method for creating new PluginInstances to ensure the proper validation of all the assets needed for correct instantiation.
